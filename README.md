# Pattern Programs

A collection of C and Java programs that generate different console-based ASCII art patterns and shapes using loops, conditions, and pattern printing techniques.

This project includes various shapes created using a combination of asterisks (*), underscores (_), and spaces. The programs demonstrate programming fundamentals such as nested loops, logic building, and structured output formatting.

## Languages

- C
- Java

## Available Patterns

- Line
- Stripe Line
- Square
- Parallelogram
- Triangle
- Reverse Triangle
- Isosceles Triangle
- Reverse Isosceles Triangle
- Hourglass
- Diamond
- Zero
- Arrow Up
- Arrow Down
- X Pattern
- Bow Tie
